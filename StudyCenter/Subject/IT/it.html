Q1) Explain any five data analysis tools

a) Consolidate is a function used to combine information from multiple sheets of the spreadsheet into one place to summarize the information. It is used to view and compare variety of data in a single spreadsheet for identifying trends and relationships
b) The Subtotal tool in Calc creates the group automatically and applies common functions like sum, average on the grouped data. It can group subtotals by using category and sorts them in ascending or descending order so that one need not to use filters.
c) The Goal Seek tool helps in finding out the input for the specific output. For example, if you want to know the number of units produced to get the desired output then use Goal seek analysis tool.
d) What-if scenario is a set of values that can be used within the calculations in the spreadsheet. A name is given to the scenario and several scenarios can be created on the same sheet. It is used to explore and compare various alternatives depending on changing conditions.
e) A solver component modifies a set of input cells in spreadsheet,observing the impact on result and constraint formula cells. the goal is used to find an optimal set of input values,for which the result is minimized,maximized or approx to target value while the constraints remain within specified limits.

Q9) Write steps to extract a table from a web page in a spreadsheet.

Step 1. Open the spreadsheet where table from web page is to be inserted.
Step 2. Select the cell to store the first cell of the table of the web page.
Step 3. Select Sheet > Link to External Data.
Step 4. The External Data dialog box is displayed. Type the URL of the source document or select it from the drop-down list if it is listed and press enter.
Step 5. A dialog box is displayed to select the language for import. Selecting Automatic shows data in the same language as in the webpage.


Q10) Write steps to register a data source that is in *.odb format.

Step 1. Select Tools > Options > LibreOffice Base > Databases. The Options - LibreOffice Base-Databases dialog box appears.
Step 2. Click the New button to open the Create Database Link dialog box.
Step 3. Enter the location of the database file, or click Browse to open a file browser and select the database file.
Step 4. Type a name to use as the registered name for the database and click OK. The database is added to the list of registered databases.

Q1) Define the terms a) Database
b) Data Redundandancy
c) Report

a) Database : A database is a collection of logically related data items stored in an organised manner.
b) Data Redundancy : In the event of requiring the same data field in several tables the data field might get repeated in number of tables. This is called as data redundancy.
c) Report : The Report feature helps to present the retrieved data in a user friendly, understandable and formatted manner.
OR
A database report is a formatted presentation of data from a database that provides structured information.

b) Form and Query
c) Network and Hierarchial data model

a) Data and Information
Data Information
Data consists of raw and unprocessed facts with no meaning wihtout context. Data which is processed to add meaningful value to it is called information.

b) Form and Query
Form Query
Forms are created for user to input information like new records or updating a value in record. Queries are used to retrieve information from that form or original table.

c)Network and Hierarchial Data model
Network Hierarchial
Relationship between records is of the parent child type. Relationship between records is expressed in the form of pointers or links.

Q10) How is redundancy or incosistency controlled in a database? Explain with an example.

Relations are set up between the tables to control data redundancy and inconsistency.
Consider Murugan managing ABC School's data. Two tables are present: Admissions and Library. 'Ram Lal Kumar' wants to change his name to 'Ram Kumar'. Without a DBMS, Murugan must update both tables manually. With a DBMS, a single update in one table automatically reflects in the other, eliminating redundancy and inconsistency.

Q12) Differentiate between one to one relationship and one to many relationship. Give suitable example to support your answer.

One to one One to many
One specific record of a master table has one and only one corresponding record in the transaction table. One specific record of the master table has more than one corresponding records in the related transaction table.
One person can have only one ticket to get entry into a stadium to view the match. One teacher can teach multiple students

Q13) Explain many to many relationship with an example.

In many-to-many relationship, there will be multiple records in the master table that correspond to multiple records in the transaction table as well.
A shopkeeper may sell multiple products to multiple customers. So many-to-many relationship exists between a product and a customer.
